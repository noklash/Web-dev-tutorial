# 01 – What is HTML? (super simple)

HTML = the **bones** of a webpage  
It says: this is a title, this is a paragraph, this is a picture...

Example – very smallest page:

```html
<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello friend!</h1>
  <p>I am learning websites.</p>
</body>
</html>
